サービスについて

Q

バク売れLPテンプレとはどのようなサービスですか？

バク売れLPテンプレは、売れる構造に基づいたLPデザインテンプレートを提供するサービスです。HTML / CSS / JavaScript ベースのテンプレートを、テキスト・画像の差し替えだけですぐにプロ品質のLPとしてご利用いただけます。

株式会社CHAINSODAが提供しています。

Q

プランは何がありますか？

2つのプランをご用意しています。

  • LP｜スタータープラン（¥55,000税込）：LPテンプレート一式（HTML/CSS/JS/画像素材）
  • LP＋バナー｜集客強化プラン（¥110,000税込）：上記＋バナーデザインテンプレート素材（PSD/Figma/PNG）
Q

テンプレートはどのような形式で提供されますか？

HTML / CSS / JavaScript ファイル＋画像素材のセットで、ZIPファイルとしてダウンロード提供いたします。特別なフレームワークやCMSは不要で、どのホスティングサービスでもすぐに公開可能です。

販売・契約について

Q

販売パートナーとしての販売フローを教えてください。

基本的な販売フローは以下の通りです。

  • ①リード獲得：見込み客へのアプローチ。LP制作に課題を感じている企業・個人がターゲットです。
  • ②契約ページへご案内：/contract/ のURLをお客様にお伝えください。プラン選択と規約同意をお客様自身で行っていただきます。
  • ③お支払い：/checkout/ でクレジットカード決済（Stripe）を行います。
  • ④テンプレート納品：決済確認後、ダウンロードリンクをメールでお届けします。
Q

支払い方法は何がありますか？

クレジットカード決済のみに対応しております（Visa / Mastercard / JCB / American Express）。Stripeを決済プラットフォームとして利用しています。銀行振込には対応しておりません。

Q

契約ページでパートナーの紹介者名（コード）は必要ですか？

はい。契約フォームに「紹介者名」の入力欄がありますので、パートナー様のお名前またはコードをお客様に入力していただくようご案内ください。これにより紹介実績が正しく記録されます。

テンプレートについて

Q

テンプレートのカスタマイズに専門知識は必要ですか？

テキストと画像の差し替えだけであれば、テキストエディタ（メモ帳やVS Code）の基本操作ができれば問題ありません。CSS変数でカラーやフォントの変更も簡単に行えます。

レイアウトの大幅変更など、より高度なカスタマイズにはHTML/CSSの基礎知識が必要です。AIチャットサポートも活用いただけます。

Q

テンプレートは複数の案件で使い回せますか？

購入されたテンプレートはご契約者様の案件でのみご利用いただけます。1契約につき1案件（1サイト）でのご利用となります。複数案件でご利用の場合は、案件数分のご契約が必要です。

第三者への共有・再配布・転売は利用規約で禁止されています。

Q

スマートフォン（レスポンシブ）に対応していますか？

はい、すべてのテンプレートがPC・タブレット・スマートフォンに対応したレスポンシブデザインとなっています。
