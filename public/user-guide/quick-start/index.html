1

テンプレートファイルを受け取る

ご契約・お支払い完了後、1営業日以内にご登録メールアドレス宛にダウンロードリンクをお送りいたします。

メールに含まれるもの

  • スタータープラン：LP テンプレート一式（HTML / CSS / JS / 画像素材）
  • 集客強化プラン：上記に加え、バナーデザインテンプレート素材（PSD / Figma / PNG）

ダウンロードしたZIPファイルを解凍し、中身を確認してください。

/* フォルダ構成の例 */ bakuure-lp-template/ ├── index.html ← メインLPファイル ├── css/ │ └── style.css ← スタイルシート ├── js/ │ └── main.js ← アニメーション・インタラクション ├── images/ ← 画像素材 └── README.md ← 詳細な説明書

ヒント: ZIPが解凍できない場合は、Mac なら「The Unarchiver」、Windowsなら「7-Zip」をお試しください。

2

テキスト・画像を差し替える

テンプレートのテキストや画像は、お手元のテキストエディタ（VS Code 推奨）で編集できます。

編集箇所の目安

  1. ヘッドライン・サブコピー：HTML ファイル内の <h1>、<h2> タグを変更
  2. CTA ボタンのテキスト：.cta-btn テキストを変更
  3. 画像差し替え：images/ 内のファイルを同名で上書き、または src="" を変更
  4. フォームの送信先：フォームの action="" を自社のフォーム処理URLに変更
  5. カラー変更：css/style.css の CSS 変数 --primary 等を変更
/* css/style.css の先頭のカラー設定 */ :root { --primary: #0852FF; ← ブランドカラーに変更 --secondary: #10B981; --bg: #FFFFFF; --text: #111827; }

注意：テンプレートの構造（HTMLタグの配置や class名）を大きく変更するとレスポンシブ対応が崩れる場合があります。まずはテキストと画像の差し替えのみを行い、動作確認してから構造変更にお進みください。

3

ローカルでプレビュー確認

編集後のLPをブラウザでプレビューして、想定通りの表示と動作かを確認します。

方法A：ファイルを直接開く

index.html をダブルクリックするだけでブラウザが開きます。ただし一部の機能（フォーム送信など）は動作しない場合があります。

方法B：ローカルサーバーを使う（推奨）

VS Code の拡張機能「Live Server」をインストールし、index.html を右クリック → 「Open with Live Server」で起動できます。

# もしくはコマンドラインから npx serve . # → http://localhost:3000 でプレビュー

チェックポイント

  • PC（デスクトップ幅）とスマホ（モバイル幅）で表示を確認
  • ボタンやリンクが正しく動作するか
  • 画像が正しく表示されるか
  • ページの読み込み速度に問題がないか
4

サーバーにアップロード（公開）

編集した LP を公開するには、テンプレートファイルをウェブサーバーにアップロードします。

おすすめの公開方法

  • Vercel（推奨）：GitHub にリポジトリを作成し Vercel と接続するだけで、自動デプロイされます。無料プランで十分対応可能です。
  • Netlify：フォルダをドラッグ＆ドロップするだけでデプロイ完了。
  • レンタルサーバー：Xserver / ConoHa 等にFTPでアップロード。
# Vercel CLI を使った例 npm i -g vercel cd bakuure-lp-template vercel # → プロジェクト設定が完了し、公開URLが発行されます

ヒント: 独自ドメインを接続すれば、よりプロフェッショナルなLPに仕上がります。多くのホスティングサービスで簡単に設定できます。

5

公開後の確認と計測設定

LPを公開したら、正常表示の確認と各種計測ツールの設定を行いましょう。

公開後チェックリスト

  1. SSL（https）の確認：ブラウザのアドレスバーに鍵マークが表示されているか
  2. OGP設定：SNSでURLを共有したときのプレビュー表示が正しいか
  3. フォーム動作テスト：テスト送信で実際にデータが届くか
  4. ページ表示速度：PageSpeed Insights でスコアを確認

計測ツールの設定

  • Google Analytics（GA4）：headタグ内にトラッキングコードを追加
  • Google Tag Manager：より柔軟なイベント計測が可能
  • Facebook Pixel / LINE Tag：広告運用時のコンバージョン計測に

ヒント: 集客強化プランをご契約の方は、バナー素材も同様にカスタマイズして広告クリエイティブとしてご活用ください。
